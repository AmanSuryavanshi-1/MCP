What is MCP?

MCP is an open standard created to solve a common problem: how to connect AI assistants to the tools and data they need to be truly helpful. Think of it as creating a common language that all AI assistants Smart programs like ChatGPT, Claude, or Gemini that can understand and respond to human language can use to talk to databases, websites, email systems, and other services.

Created by Anthropic, MCP works like a universal translator, helping AI agents AI systems that can actually do things for you (not just chat), like booking appointments, sending emails, or managing files connect safely and easily to real-world tools and information.

This standard approach makes it much easier to build AI applications that can actually help with your daily tasks, rather than just having conversations.

Key Components

  • MCP Servers Think of these as specialized service counters at a mall - each one handles specific tasks like "photo printing," "key cutting," or "phone repair." For AI, these might be "email sending," "database querying," or "file management" services. : Small programs that provide specific services (like connecting to your email or database)
  • MCP Clients Like a personal assistant who knows how to communicate with all the different service counters at the mall. The client takes your requests and figures out which service counter to visit and how to ask for what you need. : The part of your AI app that connects to and talks with the servers
  • Tools & Resources The actual services available - like "send an email," "save a file," "check the weather," or "look up customer information." These are the things your AI can actually do for you. : The actual functions your AI can perform (send emails, access files, check databases)
  • Prompts Pre-written instructions that help your AI know how to handle common situations - like having a script for "how to respond to customer complaints" or "steps for processing a refund." : Ready-made instruction templates that guide your AI's behavior

MCP Communication Flow

MCP Host/Client

The MCP Host is the application or environment where the user interacts with a Large Language Model. Examples include AI-powered IDEs like Cursor, chat interfaces such as Claude Desktop, or custom AI workflow applications.

Within the host application, the MCP Client component facilitates communication between the LLM and MCP Servers. It discovers available MCP servers, translates requests into MCP-compliant calls, and handles responses.

The MCP client acts as the "AI brain's" interface to the external world, enabling seamless leverage of external capabilities.

MCP Server

The MCP Server is a critical component that acts as a bridge between LLMs and external data sources, tools, or services. It is a lightweight program that exposes specific capabilities through the standardized MCP protocol.

Servers can be developed in any programming language that can print to stdout or serve an HTTP endpoint, with Python, JavaScript, and Go being common choices.

Resources and Adapters

Resources are file-like data that can be read by clients, encompassing API responses, file contents, database query results, or dynamically generated content. They provide contextual information to LLMs.

Adapters refer to the logic within an MCP server that translates between the MCP protocol and specific external systems. The MCP server itself acts as an adapter, bridging the LLM's world with external tools and data.

Example: A PostgreSQL MCP server adapts MCP requests into SQL queries, executes them against the database, and formats the results into MCP-compliant responses.

Messaging and Security

JSON-RPC 2.0 Messaging

MCP utilizes JSON-RPC 2.0 for messaging between clients and servers, providing a lightweight, language-agnostic, and standardized way to structure communication.

  • Requests: Bidirectional messages expecting responses
  • Responses: Results or errors corresponding to requests
  • Notifications: One-way messages requiring no response

Protocol Overview

MCP Protocol Purpose

The Model Context Protocol defines a standardized communication layer between AI applications (clients) and external services (servers), enabling secure, efficient, and scalable integrations.

  • Resource Model: URI-based resource identification
  • Tool Interface: Function-based tool definitions
  • Prompt System: Template-based prompt management

Core Capabilities

  • Resource Access: Read/write operations on external data
  • Tool Execution: Function calls with parameter validation
  • Prompt Templates: Reusable instruction patterns
  • Logging: Structured event and error tracking

Retry Logic & Fallback Strategies

Exponential Backoff Strategy
  • ‚Ä¢ Initial delay: 1 second
  • ‚Ä¢ Max retries: 3 attempts
  • ‚Ä¢ Backoff multiplier: 2x
  • ‚Ä¢ Max delay: 30 seconds
  • ‚Ä¢ Jitter: ¬±25% randomization
Retry 1: 1s ‚Üí Retry 2: 2s ‚Üí Retry 3: 4s

  • ‚Ä¢ 500 Internal Server Error
  • ‚Ä¢ 502 Bad Gateway
  • ‚Ä¢ 503 Service Unavailable
  • ‚Ä¢ 504 Gateway Timeout
  • ‚Ä¢ Network timeouts

  • ‚Ä¢ 400 Bad Request
  • ‚Ä¢ 401 Unauthorized

  • ‚Ä¢ 404 Not Found
Retry Implementation Example
import asyncio
import random
from typing import Optional

async def retry_with_backoff(
    operation,
    max_retries: int = 3,
    base_delay: float = 1.0,
    max_delay: float = 30.0,
    backoff_multiplier: float = 2.0,
    jitter: bool = True
):
    """Retry operation with exponential backoff and jitter"""
    
    for attempt in range(max_retries + 1):
        try:
            return await operation()
        except Exception as e:
            if attempt == max_retries:
                raise e
            
            # Calculate delay with exponential backoff
            delay = min(base_delay * (backoff_multiplier ** attempt), max_delay)
            
            # Add jitter to prevent thundering herd
            if jitter:
                delay *= (0.75 + random.random() * 0.5)
            
            await asyncio.sleep(delay)

# Usage example

const express = require('express');
const multer = require('multer');
const path = require('path');
const fs = require('fs').promises;

const app = express();

// Security configuration
const upload = multer({
  dest: 'uploads/',
  limits: {
    fileSize: 10 * 1024 * 1024, // 10MB limit
    files: 5
  },
  fileFilter: (req, file, cb) => {
    // Whitelist allowed file types
    const allowedTypes = /jpeg|jpg|png|pdf|txt|docx/;
    const extname = allowedTypes.test(path.extname(file.originalname).toLowerCase());
    const mimetype = allowedTypes.test(file.mimetype);
    
    if (mimetype && extname) {
      return cb(null, true);
    } else {
      cb(new Error('Invalid file type'));
    }
  }
});

// Authentication middleware
function authenticateToken(req, res, next) {
  const authHeader = req.headers['authorization'];
  const token = authHeader && authHeader.split(' ')[1];
  
  if (!token) {
    return res.status(401).json({ error: 'Access token required' });
  }
  
  jwt.verify(token, process.env.JWT_SECRET, (err, user) => {
    if (err) {
      return res.status(403).json({ error: 'Invalid token' });
    }
    req.user = user;
    next();
  });
}

// File upload tool with security
app.post('/tools/upload-file', authenticateToken, upload.single('file'), async (req, res) => {
  try {
    if (!req.file) {
      return res.status(400).json({ error: 'No file uploaded' });
    }
    
    // Generate secure filename
    const fileExtension = path.extname(req.file.originalname);
    const secureFilename = crypto.randomUUID() + fileExtension;
    const securePath = path.join('secure-uploads', req.user.id, secureFilename);
    
    // Ensure user directory exists
    await fs.mkdir(path.dirname(securePath), { recursive: true });
    
    // Move file to secure location
    await fs.rename(req.file.path, securePath);
    
    // Log the operation
    console.log(`File uploaded: ${secureFilename} by user ${req.user.id}`);
    
    res.json({
      result: {
        filename: secureFilename,
        originalName: req.file.originalname,
        size: req.file.size,
        uploadedAt: new Date().toISOString()
      }
    });
    
  } catch (error) {
    console.error('Upload error:', error);
    res.status(500).json({ error: 'Upload failed' });
  }
});

// Secure file listing
app.get('/tools/list-files', authenticateToken, async (req, res) => {

function validateInput(data, schema) {
  const errors = [];
  
  for (const [key, rules] of Object.entries(schema)) {
    const value = data[key];
    
    if (rules.required && (value === undefined || value === null)) {
      errors.push(`${key} is required`);
      continue;
    }
    
    if (value !== undefined && value !== null) {
      if (rules.type && typeof value !== rules.type) {
        errors.push(`${key} must be of type ${rules.type}`);
      }
      
      if (rules.min !== undefined && value < rules.min) {
        errors.push(`${key} must be >= ${rules.min}`);
      }
      
      if (rules.max !== undefined && value > rules.max) {
        errors.push(`${key} must be <= ${rules.max}`);
      }
    }
  }
  
  if (errors.length > 0) {
    throw new Error(`Validation failed: ${errors.join(', ')}`);

  return data;

  async readFileContent(args) {
    const validated = validateInput(args, {
      file_path: { type: 'string', required: true },
      max_lines: { type: 'number', required: false, min: 1, max: 1000 },
    });

    const { file_path, max_lines = 100 } = validated;

    try {
      const stats = await fs.stat(file_path);
      
      if (!stats.isFile()) {
        throw new Error(`Path is not a file: ${file_path}`);
      }

      // Check file size (limit to 1MB)
      if (stats.size > 1024 * 1024) {
        throw new Error('File too large (max 1MB)');
      }

      const content = await fs.readFile(file_path, 'utf-8');
      const lines = content.split('\n').slice(0, max_lines);

      const fileContent = {
        file_path,
        content: lines,
        lines_read: lines.length,
        file_size: stats.size,
        truncated: lines.length === max_lines,
        timestamp: new Date().toISOString(),
        metadata: {
          encoding: 'utf-8',
          line_endings: content.includes('\r\n') ? 'CRLF' : 'LF',
        },
      };

      return { content: [fileContent] };
    } catch (error) {
      if (error.code === 'ENOENT') {
        throw new Error(`File not found: ${file_path}`);
      }
      if (error.code === 'EACCES') {
        throw new Error('Permission denied reading file');
      }
      throw new Error(`Failed to read file: ${error.message}`);
    }
  }

  async processJsonData(args) {
    const validated = validateInput(args, {
      json_string: { type: 'string', required: true },
      operation: { type: 'string', required: true },
    });

    const { json_string, operation } = validated;
    const validOperations = ['validate', 'format', 'extract_keys', 'count_items'];
    
    if (!validOperations.includes(operation)) {
      throw new Error(`Invalid operation. Must be one of: ${validOperations.join(', ')}`);
    }

    try {
      const jsonData = JSON.parse(json_string);
      let result;

      switch (operation) {
        case 'validate':
          result = {
            valid: true,
            type: Array.isArray(jsonData) ? 'array' : typeof jsonData,
            message: 'JSON is valid',
          };
          break;

        case 'format':
          result = {
            formatted: JSON.stringify(jsonData, null, 2),
            minified: JSON.stringify(jsonData),
            size_difference: json_string.length - JSON.stringify(jsonData).length,
          };
          break;

        case 'extract_keys':
          const extractKeys = (obj, prefix = '') => {
            let keys = [];
            for (const [key, value] of Object.entries(obj)) {
              const fullKey = prefix ? `${prefix}.${key}` : key;
              keys.push(fullKey);
              if (typeof value === 'object' && value !== null && !Array.isArray(value)) {
                keys = keys.concat(extractKeys(value, fullKey));
              }
            }
            return keys;
          };

          result = {
            keys: typeof jsonData === 'object' && jsonData !== null 
              ? extractKeys(jsonData) 
              : [],
            total_keys: typeof jsonData === 'object' && jsonData !== null 
              ? extractKeys(jsonData).length 
              : 0,
          };
          break;

        case 'count_items':
          if (Array.isArray(jsonData)) {
            result = {
              type: 'array',
              count: jsonData.length,
              item_types: [...new Set(jsonData.map(item => typeof item))],
            };
          } else if (typeof jsonData === 'object' && jsonData !== null) {
            result = {
              type: 'object',
              count: Object.keys(jsonData).length,
              key_types: [...new Set(Object.values(jsonData).map(value => typeof value))],
            };
          } else {
            result = {
              type: typeof jsonData,
              count: 1,
              message: 'Single value, not a collection',
            };
          }
          break;
      }

      return {
        content: [{
          operation,
          input_size: json_string.length,
          result,
          timestamp: new Date().toISOString(),
        }],
      };

    } catch (error) {
      throw new Error(`JSON processing failed: ${error.message}`);
    }
  }

  setupErrorHandling() {
    this.server.onerror = (error) => {
      console.error('[MCP Error]', error);
    };

    process.on('SIGINT', async () => {
      console.log('\nShutting down server...');
      await this.server.close();
      process.exit(0);
    });

    process.on('uncaughtException', (error) => {
      console.error('Uncaught Exception:', error);
      process.exit(1);
    });

    process.on('unhandledRejection', (reason, promise) => {
      console.error('Unhandled Rejection at:', promise, 'reason:', reason);
      process.exit(1);
    });
  }

  async run() {
    const transport = new StdioServerTransport();
    console.error('JavaScript MCP Server starting...');
    console.error('Available tools: add_numbers, divide_numbers, get_weather_forecast, read_file_content, process_json_data');
    
    await this.server.connect(transport);
    console.error('Server running on stdio transport');
  }

# retry_patterns.py
import asyncio
import time
from typing import Callable, Any
from functools import wraps

class CircuitBreaker:
    def __init__(self, failure_threshold=5, timeout=60):
        self.failure_threshold = failure_threshold
        self.timeout = timeout
        self.failure_count = 0
        self.last_failure_time = None
        self.state = 'CLOSED'  # CLOSED, OPEN, HALF_OPEN
    
    def call(self, func):
        if self.state == 'OPEN':
            if time.time() - self.last_failure_time > self.timeout:
                self.state = 'HALF_OPEN'
            else:
                raise Exception("Circuit breaker is OPEN")
        
        try:
            result = func()
            if self.state == 'HALF_OPEN':
                self.state = 'CLOSED'
                self.failure_count = 0
            return result
        except Exception as e:
            self.failure_count += 1
            self.last_failure_time = time.time()
            
            if self.failure_count >= self.failure_threshold:
                self.state = 'OPEN'
            
            raise e

def retry_with_backoff(max_retries=3, base_delay=1, max_delay=60):
    def decorator(func):
        @wraps(func)
        async def wrapper(*args, **kwargs):
            last_exception = None
            
            for attempt in range(max_retries + 1):
                try:
                    return await func(*args, **kwargs)
                except Exception as e:
                    last_exception = e
                    
                    if attempt == max_retries:
                        break
                    
                    # Exponential backoff with jitter
                    delay = min(base_delay * (2 ** attempt), max_delay)
                    jitter = delay * 0.1 * (0.5 - asyncio.get_event_loop().time() % 1)
                    await asyncio.sleep(delay + jitter)
                    
                    print(f"Retry {attempt + 1}/{max_retries} after {delay:.2f}s")
            
            raise last_exception
        return wrapper
    return decorator

Solving the "N x M" Integration Problem

The "N times M" integration problem occurs when N different client applications need to interact with M different server-side tools or data sources. Without standardization, each client needs custom integration for each server.

  async setupAWSInfrastructure(projectName, requirements) {
    const infrastructure = {};

    // Create VPC
    const vpc = await this.aws.createVpc({

      TagSpecifications: [{
        ResourceType: 'vpc',
        Tags: [{ Key: 'Name', Value: `${projectName}-vpc` }]
      }]
    }).promise();

    infrastructure.vpcId = vpc.Vpc.VpcId;

    // Create subnets
    const subnet = await this.aws.createSubnet({
      VpcId: vpc.Vpc.VpcId,
      CidrBlock: '10.0.1.0/24',
      TagSpecifications: [{
        ResourceType: 'subnet',
        Tags: [{ Key: 'Name', Value: `${projectName}-subnet` }]
      }]
    }).promise();

    infrastructure.subnetId = subnet.Subnet.SubnetId;

    // Launch EC2 instances if required
    if (requirements.compute) {
      const instances = await this.aws.runInstances({
        ImageId: 'ami-0abcdef1234567890', // Amazon Linux 2
        MinCount: 1,
        MaxCount: requirements.instanceCount || 1,
        InstanceType: requirements.instanceType || 't3.micro',
        SubnetId: subnet.Subnet.SubnetId,
        TagSpecifications: [{
          ResourceType: 'instance',
          Tags: [{ Key: 'Name', Value: `${projectName}-instance` }]
        }]
      }).promise();

      infrastructure.instances = instances.Instances.map(i => i.InstanceId);
    }

    // Create S3 bucket for storage
    if (requirements.storage) {
      const bucket = await this.s3.createBucket({
        Bucket: `${projectName}-storage-${Date.now()}`,
        CreateBucketConfiguration: {
          LocationConstraint: process.env.AWS_REGION || 'us-west-2'
        }
      }).promise();

      infrastructure.bucketName = bucket.Location;
    }

    // Deploy Lambda functions if required
    if (requirements.serverless) {
      const lambdaFunction = await this.lambda.createFunction({
        FunctionName: `${projectName}-processor`,
        Runtime: 'nodejs18.x',
        Role: process.env.LAMBDA_EXECUTION_ROLE,

          ZipFile: Buffer.from(`

              return { statusCode: 200, body: 'Success' };
            };
          `)
        },
        Description: `Lambda function for ${projectName}`,
        Tags: {
          Project: projectName,
          Environment: 'production'
        }
      }).promise();

      infrastructure.lambdaArn = lambdaFunction.FunctionArn;
    }

    return infrastructure;
  }
